Listado de especialistas

Al buscar una cita por nombre, recordar que se esta aplicando el filtro por fecha, por esta razón solo se mostrarán las citas del día seleccionado.
Para crear una nueva cita, haz clic en "Nueva cita" y rellena el formulario.
Debes seleccionar un paciente y las especialidad. A continuación te permitirá seleccionar un especialista disponible para esa especialidad.
A continuación, el tipo de cita y la fecha y hora disponibles para esa especialidad y especialista.
Finalmente, puedes añadir un comentario adicional si lo deseas.
Para editar una cita, haz clic en el botón "Editar" correspondiente a la cita que deseas modificar. No se pueden editar citas que ya han sido realizadas.
