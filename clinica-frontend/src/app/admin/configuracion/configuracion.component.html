Configuración del Sistema

Aquí puedes ver y modificar las configuraciones del sistema. Haz clic en "Editar" para cambiar el valor de una configuración.
Al modificar ciertas configuraciones, como el color del tema, es posible que se aplique un efecto de recarga para reflejar los cambios.
Ten en cuenta que algunas configuraciones pueden requerir conocimientos técnicos para ser modificadas correctamente.
Si no estás seguro sobre una configuración, es recomendable no cambiarla sin consultar con un administrador o desarrollador.

Para configuraciones que son objetos o arrays, editarlas en formato JSON. Asegúrate de que el JSON sea válido antes de guardar.

duracion_cita: Establece la duración en minutos de cada cita. Valor numérico entero.
precio_cita: Define el precio base de una cita. Valor numérico decimal.(Para aplicar cuando se establezca una plataforma de pago)
dias_no_laborables: Lista de fechas que se consideran no laborables. Valor en formato JSON (array de strings con fechas en formato YYYY-MM-DD).
horario_laboral: Define el horario de atención de la clínica. Valor en formato JSON (objeto con días de la semana y horas de apertura/cierre).
notificaciones_email: Indica si las notificaciones por email están activadas. Valor booleano (true/false).
color_tema: Define el color principal del tema de la aplicación. Valor en formato hexadecimal (por ejemplo, #ff5733).
Crear_cita_paciente: Si está activado, los pacientes pueden crear citas desde su panel. Valor booleano (true/false).
Especialidades: Lista de especialidades médicas disponibles en el sistema. Valor en formato JSON (array de strings).
